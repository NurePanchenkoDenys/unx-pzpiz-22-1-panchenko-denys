2 ЗАВДАННЯ

     Розробити Bash-скрипт, який виконує збір базової інформації про апаратне забезпечення, операційну систему та мережеві інтерфейси і виводить її у стандартизованому форматі.

     Вимоги до скрипта:
     1) Збирати дані про CPU, RAM, материнську плату та серійний номер (Hardware).
     2) Збирати дані про дистрибутив ОС, версію ядра, дату встановлення, ім'я вузла, час роботи, кількість процесів та користувачів (System).
     3) Збирати інформацію про всі мережеві інтерфейси з IP-адресами та масками (Network).
     4) Виводити результат у стандартний потік та одночасно записувати у файл через tee.
     5) Реалізувати ротацію файлів з шаблоном YYYYMMDD-NNNN та параметром -n.
     6) Забезпечити локалізацію повідомлень та обробку помилок.
     7) Формат дати не повинен залежати від локалізації (LC_ALL=C).


3 ОПИС ВИКОНАНОЇ РОБОТИ

3.1 Загальна структура скрипта

     Скрипт має модульну структуру з окремими функціями для кожної задачі: обробка параметрів, робота з файлами, збір інформації про апаратне забезпечення, систему та мережу, форматування виводу.
     Результат виводиться одночасно у стандартний потік виведення та у файл за допомогою команди tee. Файл за замовчуванням - ~/log/task2.out.

3.2 Обробка параметрів та робота з файлами

     Обробка параметрів реалізована аналогічно до попередніх робіт: підтримка -h/--help, -n для обмеження архівів, позиційний аргумент для шляху до файлу. Ротація файлів та очищення архівів використовують ту ж логіку з шаблоном YYYYMMDD-NNNN.

3.3 Збір інформації про апаратне забезпечення (Hardware)

     CPU - функція get_cpu перевіряє наявність утиліти lscpu (Linux), файлу /proc/cpuinfo або sysctl (macOS). На macOS використовується sysctl -n machdep.cpu.brand_string.

     RAM - функція get_ram зчитує дані з /proc/meminfo (Linux) або sysctl -n hw.memsize (macOS) та переводить у мегабайти.

     Motherboard - функція get_motherboard намагається використати dmidecode (потребує sudo), файл ~/dmidecode.out як резерв, або sysctl hw.model на macOS.

     Serial Number - функція get_serial використовує dmidecode або утиліту ioreg на macOS для отримання серійного номера.

3.4 Збір інформації про систему (System)

     OS Distribution - зчитується з /etc/os-release (Linux) або через sw_vers (macOS).
     Kernel - визначається командою uname -r.
     Installation date - визначається через stat / з різними параметрами для GNU stat (Linux) та BSD stat (macOS).
     Hostname - визначається командою hostname.
     Uptime - визначається командою uptime з обробкою через sed.
     Processes - підраховуються через ps aux з підрахунком рядків.
     Users - підраховуються через команду who.

3.5 Збір мережевої інформації (Network)

     На Linux використовується ip -o addr show з обробкою через awk для виведення інтерфейсів з IP та маскою у форматі CIDR.
     На macOS використовується ifconfig з ручним перетворенням шістнадцяткової маски у CIDR-нотацію. Для кожного інтерфейсу виводиться рядок у форматі: <ім'я>: <IP>/<маска>.

3.6 Форматування виводу

     Функція collect_info збирає всі дані та форматує вивід відповідно до вимог. Дата та мітка часу Unix формуються з одного значення epoch для забезпечення відповідності одному моменту часу. Вивід передається через tee для одночасного запису у файл.

326  get_network() {
327      if command -v ip &>/dev/null; then
328          # Linux: ip addr
329          ip -o addr show 2>/dev/null | awk '{
330              iface = $2
331              if ($3 == "inet") {
332                  split($4, a, "/")
333                  printf "%s: %s/%s\n", iface, a[1], a[2]
334              }
335          }' | sort -u
336      elif command -v ifconfig &>/dev/null; then
337          # macOS: ifconfig
338          local current_iface=""
339          ifconfig 2>/dev/null | while IFS= read -r line; do
340              if echo "$line" | grep -qE '^[a-zA-Z0-9]+:'; then
341                  current_iface=$(echo "$line" | cut -d: -f1)
342              elif echo "$line" | grep -q 'inet '; then
343                  local ip mask
344                  ip=$(echo "$line" | awk '{print $2}')
345                  mask=$(echo "$line" | awk '{for(i=1;i<=NF;i++) if($i=="netmask") print $(i+1)}')
346                  if [ -n "$ip" ] && [ -n "$mask" ]; then
347                      # Перетворення hex маски у CIDR (macOS)
348                      local cidr=0
349                      if [[ "$mask" == 0x* ]]; then
350                          local hex="${mask#0x}"
351                          local bin=""
352                          for ((c=0; c<${#hex}; c++)); do
353                              local digit="${hex:$c:1}"
354                              case "$digit" in
355                                  0) bin="${bin}0000" ;; 1) bin="${bin}0001" ;; 2) bin="${bin}0010" ;; 3) bin="${bin}0011" ;;
356                                  4) bin="${bin}0100" ;; 5) bin="${bin}0101" ;; 6) bin="${bin}0110" ;; 7) bin="${bin}0111" ;;
