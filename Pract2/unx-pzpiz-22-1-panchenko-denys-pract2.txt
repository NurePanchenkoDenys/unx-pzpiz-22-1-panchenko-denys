2 ЗАВДАННЯ

     Розробити Bash-скрипт, який фіксує дату та час свого запуску, записує ці дані у файл та здійснює керування файлами журналу.

     Вимоги до скрипта:
     1) Обробляти параметри командного рядка: -h (довідка), -n num (кількість архівних файлів), file (шлях до файлу).
     2) За замовчуванням записувати дані у файл ~/log/task2.out.
     3) Якщо файл вже існує - перейменовувати його за шаблоном <ім'я>-YYYYMMDD-NNNN, де YYYYMMDD - дата, NNNN - порядковий номер.
     4) При заданому параметрі -n залишати не більше вказаної кількості архівних файлів, видаляючи найстаріші.
     5) Забезпечити локалізацію повідомлень: українською мовою при uk_UA, англійською - в інших випадках.
     6) Виводити помилки у стандартний потік помилок (stderr) та завершувати роботу з ненульовим кодом.
     7) Автоматично створювати необхідні каталоги (mkdir -p).
     8) Записувати дату у фіксованому форматі RFC 2822: Date: Sat, 07 Jul 2018 16:18:06 +0300.


3 ОПИС ВИКОНАНОЇ РОБОТИ

3.1 Загальний опис рішення

     Для виконання завдання розроблено Bash-скрипт з модульною структурою, що складається з окремих функцій для кожної задачі: визначення мови, виведення повідомлень, обробки параметрів, створення каталогів, ротації файлів, очищення архівів та запису дати.
     Скрипт підтримує три параметри командного рядка: -h або --help для виведення довідки, -n для обмеження кількості архівних файлів та необов'язковий позиційний аргумент - шлях до файлу. За замовчуванням дані записуються у ~/log/task2.out.

3.2 Обробка параметрів командного рядка

     Для обробки параметрів реалізовано функцію parse_args, яка використовує конструкцію case для розбору аргументів у циклі while. Підтримуються наступні параметри:

     -h, --help - виводить довідкову інформацію та завершує скрипт з кодом 0. Текст довідки виводиться відповідно до поточної локалізації.

     -n num - задає максимальну кількість архівних файлів. Перевіряється, що значення є цілим числом, більшим або рівним одиниці. У разі некоректного значення виводиться повідомлення про помилку.

     Позиційний аргумент - шлях до файлу для запису. Якщо передано більше одного позиційного аргументу, скрипт повідомляє про помилку.


3.3 Локалізація повідомлень

     Скрипт перевіряє значення змінних середовища LC_MESSAGES та LANG. Якщо значення починається з uk_UA, усі повідомлення виводяться українською мовою. В інших випадках - англійською. Це реалізовано у функції is_ukrainian, яка повертає код 0 для української локалізації та 1 для інших.

3.4 Робота з файлами та каталогами

     Функція ensure_directory перевіряє наявність каталогу для файлу та створює його за допомогою mkdir -p у разі відсутності. Якщо створення каталогу неможливе, скрипт виводить повідомлення про помилку та завершується.

3.5 Ротація файлів журналу

     При кожному запуску, якщо файл вже існує, він перейменовується за шаблоном:

 0  <ім'я_файлу>-YYYYMMDD-NNNN

     де YYYYMMDD - дата перейменування, а NNNN - порядковий номер з ведучими нулями (0000, 0001, 0002 тощо). Нумерація ведеться окремо для кожного дня: скрипт переглядає існуючі файли з відповідним шаблоном, знаходить максимальний номер та збільшує його на одиницю.

3.6 Керування кількістю архівних файлів

     Якщо задано параметр -n, функція cleanup_archives знаходить усі архівні файли у каталозі, сортує їх за іменем (що забезпечує хронологічний порядок завдяки формату дати у назві) та видаляє найстаріші, залишаючи не більше зазначеної кількості.

3.7 Запис дати у файл

     Дата записується у фіксованому форматі RFC 2822 за допомогою команди:

 0  LC_ALL=C date "+Date: %a, %d %b %Y %H:%M:%S %z"

     Використання LC_ALL=C забезпечує незалежність формату від поточної локалізації системи. Результатом є рядок виду:

 0  Date: Tue, 24 Mar 2026 23:40:56 +0100

3.8 Демонстрація роботи скрипта

     Перший запуск (створення каталогу та файлу):

 0  $ ./unx-pzpiz-22-1-panchenko-denys-pract2.sh
 0  Created directory: /Users/denys/log
 0  Written to /Users/denys/log/task2.out: Date: Tue, 24 Mar 2026 23:40:56 +0100

     Повторний запуск (ротація файлу):

 0  $ ./unx-pzpiz-22-1-panchenko-denys-pract2.sh
 0  Archived: /Users/denys/log/task2.out-20260324-0000
 0  Written to /Users/denys/log/task2.out: Date: Tue, 24 Mar 2026 23:41:00 +0100

     Запуск з обмеженням кількості архівів:

 0  $ ./unx-pzpiz-22-1-panchenko-denys-pract2.sh -n 2 ./logs/data.out

     Виведення довідки:

 0  $ ./unx-pzpiz-22-1-panchenko-denys-pract2.sh --help

     Обробка помилок:

 0  $ ./unx-pzpiz-22-1-panchenko-denys-pract2.sh -n abc
 0  Error: -n value must be an integer >= 1

 0  $ LC_MESSAGES=uk_UA.UTF-8 ./unx-pzpiz-22-1-panchenko-denys-pract2.sh -n abc
 0  Помилка: значення -n повинно бути цілим числом >= 1

 1  #!/bin/bash
 2  
 3  # =============================================================================
 4  # Практичне заняття №2 - Операційні системи Unix
 5  # Тема: Робота з файлами у Bash. Обробка параметрів командного рядка
 6  # Автор: Панченко Денис, гр. ПЗПІз-22-1
 7  # Опис: Скрипт фіксує дату та час запуску, записує у файл,
 8  #        здійснює керування файлами журналу (ротація)
 9  # =============================================================================
10  
11  # --- Файл за замовчуванням ---
12  DEFAULT_FILE="$HOME/log/task2.out"
13  
14  # --- Змінні ---
15  MAX_FILES=""
16  OUTPUT_FILE=""
17  
18  # =============================================================================
19  # Функція: визначення мови повідомлень
20  # Якщо LANG або LC_MESSAGES містить uk_UA - українська, інакше англійська
21  # =============================================================================
22  is_ukrainian() {
23      local lang="${LC_MESSAGES:-$LANG}"
24      if [[ "$lang" == uk_UA* ]]; then
25          return 0
26      fi
27      return 1
28  }
29  
30  # =============================================================================
31  # Функція: виведення повідомлення про помилку у stderr
32  # =============================================================================
33  error_msg() {
34      if is_ukrainian; then
35          echo "Помилка: $1" >&2
36      else
37          echo "Error: $1" >&2
38      fi
39  }
40  
41  # =============================================================================
42  # Функція: виведення інформаційного повідомлення
43  # =============================================================================
44  info_msg() {
45      if is_ukrainian; then
46          echo "$1"
47      else
48          echo "$2"
49      fi
50  }
51  
52  # =============================================================================
53  # Функція: виведення довідкової інформації
54  # =============================================================================
55  show_help() {
56      if is_ukrainian; then
57          cat <<EOF
58  Використання: $0 [-h|--help] [-n кількість] [файл]
59  
60  Опис:
61    Скрипт фіксує дату та час свого запуску та записує їх у файл.
62    Попередні файли зберігаються як архівні копії.
63  
64  Параметри:
65    -h, --help     Вивести цю довідку та завершити роботу
66    -n кількість   Максимальна кількість архівних файлів (за замовчуванням: без обмежень)
67    файл           Шлях до файлу для запису (за замовчуванням: ~/log/task2.out)
68  
69  Приклади:
70    $0                          Записати дату у ~/log/task2.out
71    $0 -n 5 ./logs/info.txt    Записати дату, залишити до 5 архівів
72    $0 --help                   Показати довідку
73  EOF
74      else
75          cat <<EOF
76  Usage: $0 [-h|--help] [-n count] [file]
77  
78  Description:
79    This script logs the current date and time to a file.
80    Previous files are preserved as archived copies.
81  
82  Options:
83    -h, --help   Show this help message and exit
84    -n count     Maximum number of archived files (default: unlimited)
85    file         Path to the output file (default: ~/log/task2.out)
86  
87  Examples:
88    $0                          Log date to ~/log/task2.out
89    $0 -n 5 ./logs/info.txt    Log date, keep up to 5 archives
90    $0 --help                   Show help
91  EOF
92      fi
93  }
94  
95  # =============================================================================
96  # Функція: обробка параметрів командного рядка
97  # =============================================================================
98  parse_args() {
99      while [ $# -gt 0 ]; do
100          case "$1" in
101              -h|--help)
102                  show_help
103                  exit 0
104                  ;;
105              -n)
106                  shift
107                  if [ -z "$1" ]; then
108                      error_msg "$(is_ukrainian && echo "параметр -n потребує значення" || echo "option -n requires a value")"
109                      exit 1
110                  fi
111                  if ! [[ "$1" =~ ^[1-9][0-9]*$ ]]; then
112                      error_msg "$(is_ukrainian && echo "значення -n повинно бути цілим числом >= 1" || echo "-n value must be an integer >= 1")"
113                      exit 1
114                  fi
115                  MAX_FILES="$1"
116                  ;;
117              -*)
118                  error_msg "$(is_ukrainian && echo "невідомий параметр: $1" || echo "unknown option: $1")"
119                  exit 1
120                  ;;
121              *)
122                  if [ -n "$OUTPUT_FILE" ]; then
123                      error_msg "$(is_ukrainian && echo "зайвий аргумент: $1" || echo "extra argument: $1")"
124                      exit 1
125                  fi
126                  OUTPUT_FILE="$1"
127                  ;;
128          esac
129          shift
130      done
131  
132      # Якщо файл не задано - використати за замовчуванням
133      if [ -z "$OUTPUT_FILE" ]; then
134          OUTPUT_FILE="$DEFAULT_FILE"
135      fi
136  }
137  
138  # =============================================================================
139  # Функція: створення каталогу для файлу
140  # =============================================================================
141  ensure_directory() {
142      local dir
143      dir="$(dirname "$OUTPUT_FILE")"
144  
145      if [ ! -d "$dir" ]; then
146          if ! mkdir -p "$dir" 2>/dev/null; then
147              error_msg "$(is_ukrainian && echo "неможливо створити каталог: $dir" || echo "cannot create directory: $dir")"
148              exit 1
149          fi
150          info_msg "Створено каталог: $dir" "Created directory: $dir"
151      fi
152  }
153  
154  # =============================================================================
155  # Функція: ротація існуючого файлу (перейменування з датою та номером)
156  # =============================================================================
157  rotate_file() {
158      if [ ! -f "$OUTPUT_FILE" ]; then
159          return
160      fi
161  
162      local base
163      base="$(basename "$OUTPUT_FILE")"
164      local dir
165      dir="$(dirname "$OUTPUT_FILE")"
166      local today
167      today="$(date +%Y%m%d)"
168  
169      # Визначення наступного номера для поточного дня
170      local max_num=-1
171      local pattern="${base}-${today}-"
172  
173      for existing in "$dir"/${pattern}*; do
174          if [ -f "$existing" ]; then
175              local num_part
176              num_part="$(basename "$existing" | sed "s/^${base}-${today}-//")"
177              if [[ "$num_part" =~ ^[0-9]+$ ]]; then
178                  local decimal_num=$((10#$num_part))
179                  if [ "$decimal_num" -gt "$max_num" ]; then
180                      max_num="$decimal_num"
181                  fi
182              fi
183          fi
184      done
185  
186      local next_num=$((max_num + 1))
187      local new_name
188      new_name=$(printf "%s/%s-%s-%04d" "$dir" "$base" "$today" "$next_num")
189  
190      if ! mv "$OUTPUT_FILE" "$new_name" 2>/dev/null; then
191          error_msg "$(is_ukrainian && echo "неможливо перейменувати файл: $OUTPUT_FILE" || echo "cannot rename file: $OUTPUT_FILE")"
192          exit 1
193      fi
194  
195      info_msg "Архівовано: $new_name" "Archived: $new_name"
196  }
197  
198  # =============================================================================
199  # Функція: видалення старих архівних файлів (якщо задано -n)
200  # =============================================================================
201  cleanup_archives() {
202      if [ -z "$MAX_FILES" ]; then
203          return
204      fi
205  
206      local base
207      base="$(basename "$OUTPUT_FILE")"
208      local dir
209      dir="$(dirname "$OUTPUT_FILE")"
210  
211      # Знаходимо архівні файли, сортуємо за іменем (за датою та номером)
212      local archives=()
213      while IFS= read -r f; do
214          archives+=("$f")
215      done < <(find "$dir" -maxdepth 1 -name "${base}-[0-9]*-[0-9]*" -type f | sort)
216  
217      local count=${#archives[@]}
218      local to_delete=$((count - MAX_FILES))
219  
220      if [ "$to_delete" -gt 0 ]; then
221          for ((i = 0; i < to_delete; i++)); do
222              rm -f "${archives[$i]}"
223              info_msg "Видалено старий архів: ${archives[$i]}" "Deleted old archive: ${archives[$i]}"
224          done
225      fi
226  }
227  
228  # =============================================================================
229  # Функція: запис дати у файл
230  # =============================================================================
231  write_date() {
232      local date_string
233      date_string="$(LC_ALL=C date "+Date: %a, %d %b %Y %H:%M:%S %z")"
234  
235      if ! echo "$date_string" > "$OUTPUT_FILE" 2>/dev/null; then
236          error_msg "$(is_ukrainian && echo "неможливо записати у файл: $OUTPUT_FILE" || echo "cannot write to file: $OUTPUT_FILE")"
237          exit 1
238      fi
239  
240      info_msg "Записано у $OUTPUT_FILE: $date_string" "Written to $OUTPUT_FILE: $date_string"
241  }
242  
243  # =============================================================================
244  # Головна частина скрипта
245  # =============================================================================
246  parse_args "$@"
247  ensure_directory
248  rotate_file
249  cleanup_archives
250  write_date
251  
252  exit 0
